# Waveshare Demo Assets

Diese Ablage fasst die wesentlichen Konstanten und Abläufe zusammen, die aus dem offiziellen Waveshare-ESP32-S3-Touch-AMOLED-1.64"-Arduino-Demo übernommen wurden (Display-QSPI, FT3267-Touch-Controller). Die eigentliche Implementierung liegt in `src/display_s3.cpp` und `src/touch_s3.cpp`, die die Demo-Schritte an LVGL anpassen.
